# Overview

This project captures shipment-ready product records using photos, metadata, and checklists. Each product is archived in OneDrive/SharePoint with a standardized folder layout and a generated Shipment Package PDF.

## MVP scope
- Tablet-first PWA
- QR-based quick access
- Multi-user roles (QC/production/admin)
- OneDrive/SharePoint as file archive
- Remote DB for metadata
