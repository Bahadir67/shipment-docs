# OneDrive Folder Standard

Base path:
`/ShipmentDocs/{YEAR}/{CUSTOMER}/{PROJECT}/{SERIAL}/`

Subfolders:
- Photos
- Docs
- Test
- Label

Recommended file naming:
- Photos: `{SERIAL}_{CATEGORY}_{INDEX}.jpg`
- Docs: `{SERIAL}_shipment-package.pdf`
- Test: `{SERIAL}_test-report.pdf`
- Label: `{SERIAL}_label.jpg`
